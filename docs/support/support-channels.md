We offer several ways to get help with FinFocus.

## Community Support

### GitHub Issues

For bug reports and feature requests, please use our GitHub Issues tracker.

- [Open an Issue](https://github.com/rshade/finfocus/issues)
- [Search Existing Issues](https://github.com/rshade/finfocus/issues?q=is%3Aissue)

### GitHub Discussions

For general questions, ideas, and show-and-tell, use GitHub Discussions.

- [Join the Discussion](https://github.com/rshade/finfocus/discussions)

## Scope of Support

FinFocus support is provided through GitHub Issues and GitHub Discussions only.
For professional support, consulting, or SLAs, please open a GitHub Discussion
with details so maintainers can advise on next steps.
